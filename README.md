Termux QEMU Pro Manager (Windows Edition)

یک اسکریپت پیشرفته و بهینه شده برای اجرای ویندوز در ترموکس با استفاده از Termux-X11 و شتاب‌دهنده گرافیکی VirGL.

ویژگی‌ها

مدیریت هوشمند منابع (RAM و CPU)

پشتیبانی از شتاب‌دهنده گرافیکی (GPU Acceleration)

نصب و مدیریت آسان فایل‌های ISO و درایورهای VirtIO

پنل گرافیکی تحت ترمینال برای مدیریت ساده‌تر

نصب سریع و آسان

برای نصب و اجرای مستقیم اسکریپت، دستور زیر را در ترموکس کپی کنید:

pkg update && pkg install git -y 



git clone [https://github.com/amirmasoud15/Qemu.git](https://github.com/amirmasoud15/Qemu.git) && cd Qemu && chmod +x win.sh && ./win.sh


پیش‌نیازها

نصب اپلیکیشن Termux-X11 (نسخه APK را از گیت‌هاب رسمی ترموکس دریافت کنید).

اجرای اسکریپت و انتخاب گزینه ۱ برای نصب پکیج‌های مورد نیاز.

نحوه استفاده

گزینه 1: اجرای ویندوز (بعد از نصب کامل).

گزینه 2: بوت از طریق ISO برای نصب اولیه ویندوز.

گزینه 3: مدیریت فایل‌ها و دانلود درایورها.

نکته: برای تجربه بهتر، در تنظیمات برنامه Termux-X11 اندروید، رزولوشن را روی Native قرار دهید.
